### tailssh
Simple script that scans for available machines with Tailscale status and creates an SSH connection.

### tailssh Install
1. Download the script
2. Install python3
3. run the script :)
4. (optional) add an alias ```echo "alias tailssh='python3 ~/tailssh.py'" >> ~/.bashrc``` then reload ```source ~/.bashrc``` now you only have to type tailssh
